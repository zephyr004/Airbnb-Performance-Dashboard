# Airbnb Listings & Reviews Analysis Dashboard

## Overview
This project is an interactive Power BI dashboard built using Airbnb listings and reviews data from multiple cities. The dashboard focuses on analyzing listing growth, city-wise performance, pricing trends, reviewer activity, and business insights through interactive visualizations and DAX measures.

The project demonstrates data cleaning, transformation, data modeling, and dashboard development skills using Power BI.

---

## Features
- Interactive Power BI dashboard
- KPI cards and trend analysis
- City-wise comparison of listings and reviews
- Airbnb growth analysis over time
- Pricing and review insights
- Dynamic filters and slicers
- Business storytelling through visualizations

---

## Key Insights
- Paris, NYC, and Sydney account for almost half of the total Airbnb listings and reviews.
- Paris has the highest number of listings and reviews among all cities analyzed.
- Airbnb reached peak listing growth in 2015 before slowing due to tightening regulations.
- Airbnb became profitable during the second half of 2016.
- The COVID-19 pandemic significantly impacted Airbnb growth trends after 2019.
- Hotel pricing differences may have influenced Airbnb demand and customer engagement.
